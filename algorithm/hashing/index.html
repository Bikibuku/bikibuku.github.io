Hashing

Some basic information about hashing

I would like to remind you that there are many different types of hashing, the one I am using is used in my course. The code is from Algorithms in C++ by Robert Segdewick.

In this type of hashing you will find array, keys and return value.

  • The integer M is the length of the array, in this case it is 17 (0-16).
  • You got two different hash types, where k is the letter number (A = 1).
  • The insert function places the letters in the array and the while loop is set if the array is not empty, explained further down.

Usage of this hash

I will be focusing mostly on exercises that are related to this course. In these exercises they often provide the length of the array and the keys.

Let’s assume that the keys are WEATHERGLASSES and the array is 17 long (0-16). I would recommend that you print out alphabet numbers, it makes it easier. The first thing you need to do is to write up the keys, and then enter the k values.

You can now, with the help of the k, calculate hash 1 and hash 2.

  • int hash1(int k) { return (k % M); }
  • int hash2(int k) { return (4 - (k % 4)); }

We have to make a table from 0 to 16 and place the hash 1 or 2 values. We always start with the hash 1 value, if the array is empty you can place the key there, if not you have to add the hash 1 and 2 value together x = (x+u) % M. The W key example: if there is a key inside the table array 6, you have to add 6+1 % 17 = 7, if it is occupied in 7 you have to run it again to you find an empty place 7+1 % 17 = 8…
